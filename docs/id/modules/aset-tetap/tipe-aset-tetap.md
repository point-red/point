# TIPE ASET TETAP


Tipe Aset Tetap merupakan pengelompokan tipe aset yang termasuk dalam akun Chart of Account

Tipe Aset Tetap memiliki fungsi sebagai pengelompokan aset tetap yang akan ditambahkan agar nominalnya dapat bertambah pada Neraca.

| Column               | Type      | Description |
| -------------------- | --------- | ----------- |
| Nama Tipe Aset Tetap | varchar   |             |
| Metode Penyusutan    | varchar   |             |
| Masa Manfaat         | int       |             |
| Tarif Penyusutan     | decimal   |             |
